# V507 V50R2 Distillation Loss Contract

## Role

V50R2 is a teacher / visual floor / reference only. The student model forward must not receive V50R2, Kinect, raw depth, or teacher points. Training code may compare model-owned student output against V50R2 teacher bank targets outside the forward pass.

## Losses

- local teacher Chamfer
- visible surface consistency
- RGB consistency
- normal consistency
- silhouette boundary
- head/hair contour
- shoulder/neck continuity
- clothing boundary
- arm/hand endpoint
- leg/foot continuity
- anti-speckle
- anti-hole
- anti-teacher-copy
- full-scene environment preservation
- hard-control separation

## Hard Gates

- Teacher copy diagnostic must fail if a candidate directly copies V50R2 teacher points/RGB.
- Loss smoke is not visual success.
- Metric improvement alone cannot be mentor-ready.
- Candidate must still pass V50R2 visual floor, full-scene insertion, same-scene controls, anti-2D, local fidelity, and manual mentor gate.
